# Mid-Level Java Interview Topics

This section targets **3–6 years of experience** positions. German companies at this level expect solid command of core Java, standard data structures, and common algorithms.

## Topics Covered

| Category | Files |
|----------|-------|

| **Data Structures** | Arrays & Strings, Linked Lists, Trees & Graphs, Stacks & Queues, Hash Maps |
| **Algorithms** | Sorting, Searching, Basic Dynamic Programming, Recursion |
