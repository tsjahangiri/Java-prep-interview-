# 8-Week Daily Practice Schedule

## Overview

| Week | Focus |
|------|-------|
| 1–2 | Core Java fundamentals + easy/medium DSA |
| 3–4 | Data structures deep-dive + medium DSA |
| 5–6 | Advanced algorithms + system design basics |
| 7 | Senior Java + hard problems |
| 8 | Mock interviews + review + behavioral prep |

---

## Week 1: Java Foundations + Arrays/Strings

### Monday — Wednesday (Theory + Easy Problems)
**Morning (45 min):**
- Read `mid-level/core-java/05-oop-principles.md`
- Read `mid-level/core-java/01-collections-and-generics.md`

**Evening (45 min):**
- Solve 2 easy array problems (e.g., Two Sum, Valid Anagram)
- Time yourself: 15 minutes per problem
- Write the solution from memory after reading the explanation

### Thursday — Friday (Medium Problems)
**Daily (60 min):**
- 2 medium problems from `mid-level/data-structures/01-arrays-and-strings.md`
- Focus: sliding window and two-pointer patterns

### Weekend (Review)
- Attempt the problems again without looking at solutions
- Write down 3 things you still struggle with

---

## Week 2: Collections, Streams, Linked Lists

### Monday — Tuesday
- `mid-level/core-java/03-java8-streams-lambdas.md`
- Stream exercises: filter/map/groupBy/flatMap practice

### Wednesday — Thursday
- `mid-level/data-structures/02-linked-lists.md`
- Implement: reverse, detect cycle, merge sorted lists from memory

### Friday (Mock Coding Session)
- Pick 2 problems you haven't seen before
- Time yourself: 25 minutes each, no hints
- Review after

---

## Week 3: Trees & Graphs + Stacks/Queues

### Focus Problems
- Binary tree traversals (all 4)
- Validate BST
- Number of Islands (DFS + BFS versions)
- Valid Parentheses
- Daily Temperatures
- K-th Largest Element

### Daily Pattern
```
Day 1: Read theory → solve 1 problem
Day 2: Solve 2 new problems
Day 3: Revisit Day 1 problems from memory
Day 4: Solve 2 new problems
Day 5: Full mock session (3 problems, 30 min each)
```

---

## Week 4: Algorithms — Sorting, Searching, DP Basics

### Goals
- Implement merge sort and quick sort from memory
- Binary search and all variants
- Fibonacci DP in all 4 approaches
- Knapsack (2D and 1D)
- LCS

### Spaced Repetition Rule
After solving a problem:
- Review it 1 day later
- Review it 3 days later
- Review it 1 week later
- If you can solve it in < 15 min without hints, you own it

---

## Week 5: Senior Java + Advanced Algorithms

### Topics
- `senior-level/advanced-java/01-jvm-internals.md`
- `senior-level/advanced-java/02-concurrency-advanced.md`
- `senior-level/algorithms/01-dynamic-programming-advanced.md`
- Edit Distance, Word Break, Coin Change

### Mock Interview Format (Friday)
1. Solve 1 easy + 1 medium + 1 hard (90 min total)
2. Explain each solution aloud as you code
3. Record yourself (optional — very helpful for reviewing your communication)

---

## Week 6: System Design

### Daily (60 min)
- Read 1 system design file per day
- Draw the architecture on paper
- Practice the 6-step framework from `system-design/README.md`

### Design Practice Problems
1. Design a URL shortener
2. Design a notification system
3. Design a rate limiter
4. Design Twitter's timeline
5. Design a distributed cache

---

## Week 7: Hard Problems + Design Patterns

### Hard LeetCode-style Problems
- Burst Balloons (interval DP)
- Word Ladder (BFS)
- N-Queens (backtracking)
- Dijkstra's shortest path
- Topological sort

### Design Patterns
- Implement: Singleton, Builder, Decorator, Strategy, Observer
- Discuss trade-offs for each

---

## Week 8: Mock Interviews + Behavioral

### Monday — Wednesday: Full Mock Interviews
- 1-hour sessions simulating real interviews
- 1 coding problem + 1 system design + 1 behavioral question
- Ask a friend to interview you, or use platforms like Pramp, interviewing.io

### Thursday — Friday: Behavioral Preparation
- Prepare 10 STAR stories (see `senior-level/behavioral/01-star-method.md`)
- Practice aloud — fluency matters
- Review `practice-strategy/04-german-interview-culture.md`

### Weekend: Final Review
- Go through your "struggled with" notes from all 8 weeks
- Review the 10 problems you found hardest
- Rest and confidence prep

---

## Daily Time Budget

| Commitment | Suggested Daily Time |
|------------|---------------------|
| Intensive prep (8 weeks out) | 90 min/day |
| Moderate prep (4 weeks out) | 2 hrs/day |
| Final week | 3 hrs/day + rest |

**Important**: Consistency beats intensity. 45 minutes every day beats 6 hours on weekends.
