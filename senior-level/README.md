# Senior-Level Java Interview Topics

This section targets **6+ years of experience** positions. Senior Java roles in Germany emphasize depth in JVM internals, production-grade concurrency, system design, and leadership/communication skills.

## Topics Covered

| Category | Files |
|----------|-------|
| **Advanced Java** | JVM Internals, Advanced Concurrency, Design Patterns, Performance Tuning |
| **Hard Algorithms** | Advanced DP, Graph Algorithms, Complex Data Structures |
| **Behavioral** | STAR Method, Common Senior Behavioral Questions |

## What Sets Senior Interviews Apart

1. **Depth over breadth**: expect follow-up questions that go 3–4 levels deep
2. **System-level thinking**: "how would this behave in production?" / "what breaks at scale?"
3. **Trade-off discussions**: there is rarely one right answer; articulate pros and cons
4. **Code quality**: clean code, meaningful names, SOLID principles in live coding
5. **Ownership mindset**: talk about decisions you drove, teams you mentored, incidents you resolved

## Interview Frequency by Topic

- 🔴 **Very Common**: Design Patterns, JVM memory model, Advanced concurrency, Hard DP
- 🟠 **Common**: Graph algorithms (Dijkstra, topological sort), custom data structures
- 🟡 **Occasional**: GC tuning, JVM bytecode, lock-free data structures
