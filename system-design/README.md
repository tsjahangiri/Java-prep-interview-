# System Design

System design questions assess your ability to design large-scale distributed systems. Senior Java developers in Germany are expected to have practical experience with these topics.

## How to Approach a System Design Interview

### The 6-Step Framework (45 min session)

| Time | Step |
|------|------|
| 0–5 min | **Clarify requirements** — functional + non-functional |
| 5–10 min | **Estimate scale** — QPS, storage, bandwidth |
| 10–15 min | **High-level design** — draw core components |
| 15–30 min | **Deep-dive** on 2-3 critical components |
| 30–40 min | **Address bottlenecks** — scale, reliability, consistency |
| 40–45 min | **Discuss trade-offs** and alternatives |
